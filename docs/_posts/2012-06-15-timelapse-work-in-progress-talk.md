Recently, I've given several talks about the current status of the
Timelapse project in Silicon Valley and in other places. The talk
covers the main contributions to date:

 * How VM record/replay techniques can be adapted for browser architectures
 * Timelapse, our prototype infrastructure and tool implemented in
   WebKit
 * Design concepts and formative evaluation of a developer tool for
   using recordings during debugging and other tasks
   
The slides are posted on
[Slideshare](http://www.slideshare.net/brrian/timelapse-interactive-recordreplay-for-the-web). My
presentation at Mozilla was recorded, and is viewable on [Air Mozilla](https://air.mozilla.org/timelapse/). Feel free to ask questions or let
me know what you think!
